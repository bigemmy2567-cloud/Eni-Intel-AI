# ✨ Overview

Eni Intel AI is an intelligent personal AI assistant designed specifically for frontend developers.

Instead of simply generating code, Eni understands design principles, user experience, accessibility, responsiveness, animations, and frontend architecture to help developers build professional applications faster.

Whether you're building:

- SaaS platforms
- Landing pages
- Dashboards
- AI applications
- E-commerce websites
- Portfolio websites
- Mobile responsive interfaces

Eni helps you go from idea to production.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/bigemmy2567-cloud/Eni-Intel-AI.git
```

Move into the project

```bash
cd Eni-Intel-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
